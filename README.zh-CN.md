### 清晰的目录结构

* App
    * Http
        * Controller
            * Admin
                * IndexController
                * UserController
                * ConfigController
                * ...
        * Request
            * Admin
                * Index
                    * StoreRequest
                    * UpdateRequest
                    * DestroyRequest
                * User
                    * ...
                * Config
                    * ...
                * Request.php
    * Models 
        * User
            * User.php    
            * UserExt.php
            * UserMessage.php
        * Config
            * Config.php
            * ...
        * BaseModel.php
    * Repositories (目录结构应与model一致,结构清晰)
        * User
            * UserRepository.php
            * UserExtRepository.php
            * UserMessageRepository.php
        * ...
